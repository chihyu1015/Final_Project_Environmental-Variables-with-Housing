# Final Project

This repository contains the full Python workflow for the final project,
including data cleaning, feature engineering, exploratory spatial analysis,
and model training.

## Project Structure

- `report/`  
  Final project report (PDF)

- `step1_data_cleaning/`  
  Data preprocessing and cleaning scripts

- `step2_feature_engineering/`  
  Feature construction and outlier handling

- `step3_exploratory_analysis/`  
  Exploratory spatial and statistical analysis

- `step4_model_training/`  
  Model training, tuning, and evaluation

## Requirements
Python packages used in this project are listed in `requirements.txt`.
